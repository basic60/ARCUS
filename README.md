# ARCUS
## Description
ARCUS is a simple os kernel based on the x86-64 architecture.

## Feature
- A simple bootloader without using grub
- Memory management

## Requirements
- x86-64 GCC coross-compier
- NASM
- GNU Make
- Binutils
- Qemu 5.2.0+

## Build & Run
```
$ cd src && make all
```
